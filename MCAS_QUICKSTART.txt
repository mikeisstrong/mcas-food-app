================================================================================
MCAS FOOD ASSESSMENT SYSTEM - QUICK START
================================================================================

This is an AI-powered MCAS/Histamine Intolerance food assessment tool built on:
- SIGHI Database (400+ foods with compatibility ratings)
- OpenAI GPT-4 (intelligent assessment of unknown foods)
- Flask Backend (Python API)
- Modern Frontend (HTML/JavaScript with responsive design)

================================================================================
SETUP (2 minutes)
================================================================================

1. INSTALL DEPENDENCIES:
   pip install -r requirements-mcas.txt

2. VERIFY .env FILE EXISTS:
   Check that .env contains OPENAI_API_KEY (already added)

3. START THE API SERVER:
   python mcas_food_api.py

   Expected output:
   * Running on http://localhost:5000

4. OPEN THE FRONTEND:
   Open mcas_app.html in your web browser
   - Or: open mcas_app.html (macOS)
   - Or: Double-click the file in finder/explorer

================================================================================
USAGE
================================================================================

SEARCH & ASSESS:
1. Type a food name (e.g., "chicken breast", "tomato sauce", "blue cheese")
2. Click "Assess" or press Enter
3. Get instant results with:
   - SIGHI rating (0-3)
   - AI assessment with confidence
   - Reaction probability percentage
   - Mechanisms involved (H, H!, A, L, B)
   - Preparation recommendations
   - Similar foods in database

BROWSE DATABASE:
1. See 400+ foods from SIGHI
2. Filter by rating (0 = safe, 3 = avoid)
3. Click any food to analyze it
4. View database statistics

================================================================================
RATING SCALE
================================================================================

0  - SAFE: Well tolerated, eat normally
1  - CAUTION: Minor symptoms possible, test in small amounts
2  - INCOMPATIBLE: Significant symptoms likely, avoid
3  - VERY POOR: Severe symptoms expected, strictly avoid

================================================================================
WHAT IT ANALYZES
================================================================================

H!  = Highly perishable (fresh meat)
H   = High histamine (aged cheese, fermented foods)
A   = Biogenic amines (citrus, chocolate, nuts)
L   = Histamine liberators (tomatoes, strawberries, shellfish)
B   = DAO inhibitors/blockers (fermented, alcohol)

API ENDPOINTS (for advanced use)
================================================================================

  Input: {"food_name": "chicken"}
  Returns: SIGHI data + LLM assessment

GET /api/search-foods?q=tomato
  Returns: Foods matching query

GET /api/foods-by-rating?rating=2

GET /api/foods-by-category
  Returns: Foods grouped by category

GET /api/database-stats
  Returns: Database statistics

================================================================================
TROUBLESHOOTING
================================================================================

ERROR: "Cannot connect to API"
FIX:   1. Make sure python mcas_food_api.py is running
       2. Check port 5000 is not blocked
       3. Verify API key in .env

ERROR: "OpenAI API Error"
FIX:   1. Verify API key in .env is correct
       2. Check you have GPT-4 access
       3. Ensure you have API credits

ERROR: "No results found"
FIX:   1. Try different spelling/name
       2. Browse database to find similar food
       3. Try more specific name (e.g., "cherry" not just "fruit")
