TROUBLESHOOTING
================================================================================

"Cannot connect to API"
  ‚Üí Make sure you ran: python mcas_food_api.py
  ‚Üí Check that the API is still running in your terminal

"OpenAI API Error"
  ‚Üí Verify API key is in .env
  ‚Üí Check you have GPT-4 access at openai.com

"Nothing is loading"
  ‚Üí Hard refresh browser: Cmd+Shift+R (macOS) or Ctrl+Shift+R (Windows)
  ‚Üí Check that port 5000 is not being used by another app

"No results found"
  ‚Üí Try different spelling
  ‚Üí Browse database to find similar foods
  ‚Üí Try with quotes: "blue cheese" not "bluecheese"

HOW IT WORKS
================================================================================

SIMPLE VERSION:
  1. You type a food name
  2. Backend searches SIGHI database
  3. If found: returns rating
  4. If not found: asks OpenAI GPT-4
  5. GPT-4 analyzes using SIGHI guidelines
  6. Returns assessment with confidence
  7. Browser displays results
